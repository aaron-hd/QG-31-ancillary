# QG-31-ancillary
Ancillary files for arXiv publication 2306.04725

depedencies:
Mathematica 12
xAct package (http://www.xact.es/) as of 2020

Both notebook files Sec-II.nb and Sec-III.nb can be executed from top to bottom (without executing initialization cells) and reproduce expressions presented in the paper.

Comments indicate which expressions are to be compared to which equations in the paper.
